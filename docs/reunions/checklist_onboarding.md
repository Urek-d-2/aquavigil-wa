## Comment inviter un collaborateur (rappel M1)
1. Repo → **Settings** → **Collaborators** → **Add people**
2. Saisir le `@username` ou l'email → choisir le rôle **Write**
3. Le membre reçoit une invitation par email → doit l'accepter
4. Pour le kanban : ouvrir le projet → **⋯** → **Settings** → **Manage access** → ajouter le membre
